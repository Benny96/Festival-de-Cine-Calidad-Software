Descripción detallada

Este bloque de código provee una API RESTful para establecer una conexión entre la parte cliente y servidora de la aplicación. A su vez, tiene acceso a los diferentes tipos de dato utilizados en la aplicación representados con el patrón de diseño DTO para poder serializarse y transmitirse, además de al DAO para gestionar la base de datos MySQL con DataNucleus.
