Descripción detallada

Este bloque de código representa un conjunto de pruebas unitarias con el objetivo de validar que las clases DTO se generan correctamente, tanto en su contenido como en el proceso de creación (p. ej. para añadir un ComentarioDTO a una PeliculaDTO). A estas pruebas unitarias se les ha añadido tests de rendimiento para validar que su generación es rápida.
