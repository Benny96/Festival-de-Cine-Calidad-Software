Descripción detallada

Es el paquete que contiene las clases referidas al nivel servidor de la aplicación desarrollada. Este paquete contiene una capacidad a acceso de datos MySQL mediante DataNucleus (DAO), diversos tipos de datos requeridos para que la aplicación funcione (Data), y un FestivalCineManager, quien aporta una API RESTful para que la comunicación Cliente-Servidor se pueda ejecutar.
