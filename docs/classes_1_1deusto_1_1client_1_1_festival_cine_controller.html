◆ generateFixtures()

void es.deusto.client.FestivalCineController.generateFixtures ( )

Este método permite generar los datos de partida para que la aplicación pueda operar con algo. Además, cubre una función importante, que es la de generar el usuario "admin", el cual es el único que puede crear más películas y actores dentro del sistema.
